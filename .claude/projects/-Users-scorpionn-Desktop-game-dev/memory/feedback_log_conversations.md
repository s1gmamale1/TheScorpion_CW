Log every conversation exchange to a session log immediately after each response.

**Why:** User is on a tight deadline and cannot afford to lose context if a session drops or gets compressed. They want continuity across conversations.

**How to apply:** After every response, append a summary of what was discussed/decided/done to the active session log file in memory. Include decisions made, questions asked, answers given, and any action items.
